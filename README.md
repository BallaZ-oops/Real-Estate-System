# Real Estate System - Backend

REST API для системы управления недвижимостью.

## Технологии
- Java 17
- Spark Framework
- H2 Database
- JWT
- BCrypt

## Запуск

```bash
mvn clean compile exec:java -Dexec.mainClass=com.realestate.Main

Сервер запустится на http://localhost:8080

text
